## 词库配置文件
下载 [配置文件](https://github.com/liuour/rime/archive/master.zip) 解压出【配置文件】。

点选**用户文件夹**打开 Rime 目录文件夹，将【配置文件】里所有文件粘贴进去，并选择覆盖。

> 注意：覆盖前建议先备份原文件。

点击**重新部署**完成配置。

> 注1：修改任何配置文件都要**重新部署**才会生效（快捷键 `Control+Option+｀`）。
>
> 注2：搜狗词库较大，重新部署时间会稍长，请耐心等待。
